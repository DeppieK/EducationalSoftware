Line Breaks

We can separate lines of text in HTML with the help of the line break tag: <br>.

For example:

<h1> My favorite things </h1>
<p>
Raindrops on roses <br>
Whiskers on kittens <br>
Bright copper kettles <br>
Warm woolen mittens
</p>

Links

To create a link in HTML, we use the anchor tag <a> with the href attribute.

For example:

<a href="https://example.com">Link Text</a>

To link the text to a webpage, we add <href=""> along with a uniform Resource Locator (URL) <href> id short for "hypertext reference.

For example:

<a href="hhtps://html.holic.gr">
Learn to Code
</a>

Attributes

<href> is an attribute. All attributes have two things in common: they provide extra information and they go inside the opening tag.
Attrobutes are added after the name of the tag, and before the > closing sign.

For example:

<a href="hhtps://html.holic.gr">
Learn to Code
</a>

Images

To add image to a website, we start with <img> tag.

Just like <br>, <img> is an empty tag. That means it has no closing tag.

To display an image, an image tag needs the <src> attribute. <src> stands for source.

For example:

<h3>Drawing of
<em> Earthrises </em> 
</h3>
<img src="https://html.holic.gr/app.png">

Including JavaScript

To add JavaScript code to a webpage, we start with the <script> and </script> tags.

The <script> element is the last element before the closing </body> tag in the HTML structure.

In the JS file, we can write JavaScript like <const username = "Lee">

For example:

index.html
<body> 
<h3>Tweeter</h3>
<script src="script.js"></script>
</body>
script.js
const username= "Lee";

Buttons

To create a button users can click on, we need the opening <button> and closing </button> tags.

For example:

<body>
<p> Subscribe to our Newsletter </p> 
<button> Subscribe </button>
</body>

We can make texts appear on a button by typing them between the tags.

For example:

<body>
<h1> Facebook</h1>
<p> Log in to your account to get started </p>
<button> Login </button>
</body>

Input Elements

We create input elements using the empty <input> tag.

For example:

<body>
<h4> Sign up </h4> 
<input/>
</body>

The default value for type is <text> which lets users enter in text they can see.

For example:

<body>
<h4> Sign up </h4>
<input type="text" placeholder="Email"> <br>
<input type="password" placeholder="Password"> </body>

Divs

We create a group of elements, we surround the elements with <div> and </div> tags.

For example:

<body>
<div>
<h3> Latest News </h3> 
<h1> Where VC's are investing in Fintech </h1>
<p> Arman</p>
</div>
</body>

Ordered Lists

An ordered list is made out of <ol> and </ol> tags.The letter O stands for "ordered" and the letter L stands for "list".

For example:

<body>
<ol></ol>
</body>

To add a list item, we add text between the <li> and </li> tags.In this case the L stands for "list" and the I stands for "item".

For example:

<body>
<ol>
<li>Tortoise</il>
</ol>
</body>

Unordered Lists

To create an unordered list, we use the <ul> and </ul> tags.The letter U stands for "unordered" and the letter L stands for "list".

For example:

<body>
<ul></ul>
</body>

To add a list item, we add text between the <li> and </li> tags.In this case the L stands for "list" and the I stands for "item".

For example:

<body>
<ol>
<li>Home</il>
<li>About</il>
<li>Contact</il>
</ol>
</body>

Linking Local webpages

We can access locan webpages like trail.html by adding </> before the file name, just like locan images.

For example:

index.html
<body>
<h5> History of the American Pioneers</h5> 
<a href="/trail.html>
Follow the 
 <strong> trail </strong>
 by clicking here </a>
</body>
