Unit 1: What is HTML?

HTML, short for HyperText Markup Language, is the core language of the World Wide Web. It defines the content and basic structure of web pages and web applications.

What is Markup?

“Markup” refers to how HTML “marks up” the page with annotations within the HTML file. These annotations tell the browser how to display the document to visitors.

HTML: Language or Not?

Whether HTML is a “programming” language is debated. Most agree it's a “markup” language. Some argue it’s a declarative programming language.
